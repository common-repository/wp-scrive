== Description ==

WP Scrive by Webbstart is an unofficial Scrive integration that make the integration to Scrive much easier.

The standard form template are fully customizable and replacable.

**Docs and support**

We're currently working on creating documentation for the plugin. Right now we'll just ask you to be patient.
You can use the [WordPress support forum](https://wordpress.org/support/plugin/wp-scrive/) for help with installing and configuring the plugin.

We offer premium support to all paying [Webbstart](https://www.webbstart.nu/) customers.

== Frequently Asked Questions ==

= How do I install it? =

After you have enabled the plugin you need to go to the plugin settings(under settings) and fill in your Scrive credentials.

= Can I help translate it to my language? =

Contribute to the translation of this plugin at [Translating WordPress](https://translate.wordpress.org/projects/wp-plugins/wp-scrive/).

= I do miss one feature =

If you would like to suggest any feature to be added, please write an email to our project manager [jamie.blomerus@webbstart.nu](mailto:jamie.blomerus@webbstart.nu?subject=Feature%20suggestion%20-%20WP%20Scrive%20by%20Webbstart).
